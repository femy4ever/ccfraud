Welcome to Credit Card Fraud Detection

This web app helps detect credit card fraud. Here's how it works:

Step 1: Make sure to have downloaded the repo. Then open a terminal window, Navigate to the directory where app.py is located and run locally.

Step 2: Now when it starts running, Use the link from the terminal (localhost:5000 or http://127.0.0.1:5000), open in a browser and Input your data in the fields provided, Then click Predict below.

Step 3: Once the prediction prompt opens with the prediction result, close or reset to repeat process.
